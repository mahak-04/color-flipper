# Color Flipper

A fun and interactive web application that allows users to change the background color of the page with the click of a button. Each click generates a new random hexadecimal color.

## Features

* **Random Color Generation:** Dynamically generates and applies a new random hexadecimal color to the background.
* **Color Display:** Shows the current background color's hexadecimal code.
* **Interactive Button:** A clear button to trigger the color change.
* **Smooth Transitions:** CSS transitions make the color changes visually appealing.

##  Technologies Used

* **HTML5:** Structure of the application (button, text display).
* **CSS3:** Styling, layout, and smooth background color transitions.
* **JavaScript (ES6+):** Core logic for generating random colors, updating the display, and manipulating the `body`'s background style.

## Getting Started

To run this project locally, simply follow these steps:

1.  **Clone the Repository (or download the files):**
    ```bash
    git clone [YOUR-REPOSITORY-LINK-HERE]
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd color-flipper
    ```
3.  **Open the file:**
    Open the `index.html` file in your preferred web browser (e.g., Chrome, Firefox) by double-clicking it.

## Key JavaScript Concepts Demonstrated

This project is excellent for learning about:

* Manipulating **DOM styles** (`element.style.backgroundColor`).
* Implementing **Event Listeners** (`.addEventListener()`) for user interactions.
* Creating **functions** for modular code (`getRandomHexColor`, `getRandomNumber`).
* Working with **arrays** and basic **random number generation**.
